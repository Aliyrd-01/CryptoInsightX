# PHP Backend для CryptoInsightX

Этот проект содержит PHP (Laravel) бэкенд, который заменяет Node.js версию для развертывания на Hostinger.

## Структура

- `app/Http/Controllers/AuthController.php` - контроллер для авторизации (register, login, logout, me)
- `app/Models/User.php` - модель пользователя
- `routes/api.php` - API маршруты
- `config/` - конфигурационные файлы
- `DEPLOYMENT.md` - подробная инструкция по развертыванию

## API Endpoints

Все эндпоинты доступны по префиксу `/api`:

- `POST /api/auth/register` - регистрация нового пользователя
- `POST /api/auth/login` - вход в систему
- `POST /api/auth/logout` - выход из системы
- `GET /api/auth/me` - получение текущего пользователя

## Особенности

- Использует PHP сессии для авторизации (совместимо с фронтендом)
- Подключение к существующей MySQL базе данных
- CORS настроен для работы с фронтендом
- Готово к расширению для админки и отчетов

## Следующие шаги

1. Следуйте инструкциям в `DEPLOYMENT.md` для развертывания на Hostinger
2. После успешного развертывания можно удалить папку `server/` (старый Node.js бэкенд)
3. Для добавления админки и отчетов можно расширить существующую структуру
